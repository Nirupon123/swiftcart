# swiftcart
This repository is not just a standard CRUD application; it is a deep exploration into distributed system design and high-availability backend engineering. Designed as an end-to-end clone of platforms like Blinkit or Instacart, this project tackles the brutal realities of quick-commerce: handling flash-sale concurrency without overselling, real-time geospatial routing, and asynchronous event processing.

Anchored by a Django core, the architecture is built to explore industry-standard scalability constraints, including horizontal scaling, distributed caching (Redis), event-driven microservices (Kafka/RabbitMQ), and autonomous observability (AIOps).
